## Description

[Nest](https://github.com/nestjs/nest) Application for monitoring purpose, 
Monitoring of NestJs applications, on app shut down catch event where you can do what ever you want, send some notification, email or SMS where you wish.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
